Contains the python code to generate synthetic data generators using Synthetic Data Vault. 
Requires SDV 0.17.1
Datasets for testing are used from Kaggle
Please have the sdv_synth_generator under the directory by name SyntheticDataVault, anomaly_detection, malware_detection and GeneratorEvaluation are placed under the directory by name Evaluation.
The datasets are placed under the folder by name Real Datasets and the generated output are saved under folder Synthetic Datasets
